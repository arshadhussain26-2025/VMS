### Issue 1: "I don't see the Save as PDF option"
**Solution:** 
- Make sure you're using **Chrome** or **Edge** browser
- In the print dialog, look for **"Destination"** dropdown
- Click it and select **"Save as PDF"** or **"Microsoft Print to PDF"**

### Issue 2: "PDF looks wrong - no colors"
**Solution:**
- In the print dialog, find **"More settings"**
- Enable **"Background graphics"** checkbox
- This includes colors and styling

### Issue 3: "Word won't open the HTML file"
**Solution:**
- Make sure you change the file type filter to **"All Files"** or **"Web Pages"**
- Or save the HTML file with a .html extension if it doesn't have one
- Try opening in Word 2016 or later (older versions might not work well)

### Issue 4: "File is too large"
**Solution:**
- The HTML file is only ~60KB, very small
- If PDF is large after adding screenshots, compress the screenshots first
- Use online tools like TinyPNG.com to compress images
